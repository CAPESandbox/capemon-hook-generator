# CAPEv2 (capemo) hook(s) generator

This script attempts to automatically generate new hooks (hookdefs) for capemon, the CAPEv2 monitor. The hooks are generated according to the info specified in [this repo](https://github.com/RazviOverflow/capemon#how-to-add-hooks-to-capemon).



Changes to `config.ini` are ommited with `git update-index --skip-worktree config.ini`.
